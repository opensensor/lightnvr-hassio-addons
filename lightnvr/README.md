# LightNVR

![Supports amd64][amd64-shield]
![Supports aarch64][aarch64-shield]

Lightweight Network Video Recorder for Home Assistant — go2rtc streaming,
WebRTC/HLS live view, ONVIF discovery, motion/object detection and continuous
recording, wrapping the official `ghcr.io/opensensor/lightnvr` image.

See [DOCS.md](DOCS.md) for installation, configuration, networking and storage
details.

Default login is `admin` / `admin` — change it immediately after first start.
